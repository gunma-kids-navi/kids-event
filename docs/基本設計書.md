# 基本設計書

**プロジェクト名**: ぐんまこどもイベント（gunma-kids-events）  
**バージョン**: 1.0.0  
**作成日**: 2026年5月16日

---

## 1. システム概要

### 1.1 目的

群馬県内で開催される子ども向けイベントの情報を一元集約し、Web ブラウザ上で閲覧できる静的 Web アプリケーション。

### 1.2 対象ユーザー

群馬県内在住または訪問予定の子育て世帯。

### 1.3 システム形態

- SPA（シングルページアプリケーション）
- 静的ファイル（HTML / CSS / JavaScript）として配信
- サーバーサイド処理なし（イベントデータはビルド時に静的ファイルへ埋め込み）

## 2. 機能一覧

### 2.1 ホーム画面（HomeView）

| 機能                 | 内容                                                                                                           |
| -------------------- | -------------------------------------------------------------------------------------------------------------- |
| 統計表示             | 開催中・近日開催・総掲載件数を表示                                                                             |
| 開催中イベント表示   | ステータスが `ongoing` のイベントを最大3件カード形式で表示                                                     |
| 近日開催イベント表示 | ステータスが `upcoming` のイベントを最大3件カード形式で表示                                                    |
| エリア一覧表示       | EVENTS データから一意なエリア名を抽出し、エリアごとの件数を表示                                                |
| エリアリンク         | エリアカードをクリックすると `/events?area=<エリア名>` へ遷移（EventsView のエリアフィルターが自動適用される） |

### 2.2 イベント一覧画面（EventsView）

| 機能                 | 内容                                                                               |
| -------------------- | ---------------------------------------------------------------------------------- |
| ステータスフィルター | すべて / 開催中 / 近日開催 / 終了                                                  |
| カテゴリフィルター   | すべて / 体験・工作 / 展覧会 / 自然・アウトドア / 文化・学習 / 祭り・フェスタ      |
| 情報源フィルター     | SOURCES 配列の各情報源をチップで複数選択可（OR 条件）。未選択時は全件表示          |
| エリアフィルター     | すべて + EVENTS から抽出した一意エリア名                                           |
| キーワード検索       | title・area・venue・tags・label・desc を対象に部分一致検索（大文字小文字区別なし） |
| ソート               | 日付が早い順 / 日付が遅い順 / タイトル順                                           |
| 結果件数表示         | フィルター適用後の件数を表示                                                       |
| URL パラメータ連携   | `?area=xxx` クエリパラメータをエリアフィルターに反映                               |
| イベントカード       | EventListItem コンポーネントで表示、クリックでモーダル表示                         |
| 空状態表示           | 条件一致イベントがない場合にメッセージ表示                                         |

### 2.3 カレンダー画面（CalendarView）

| 機能               | 内容                                                                                         |
| ------------------ | -------------------------------------------------------------------------------------------- |
| 月次カレンダー表示 | 日〜土の週並び、日付マス表示                                                                 |
| 月移動             | 前月・次月ボタン（次月は TODAY+6ヶ月、前月は TODAY-2ヶ月を上下限とする）                     |
| 祝日ハイライト     | `isHoliday()` 関数で判定した日付をスタイル区別                                               |
| 今日ハイライト     | 今日の日付を視覚的に区別                                                                     |
| イベントドット     | 各日付マスに最大2件のイベントタイトル（8文字）を表示、3件以上は「+N」表示                    |
| 日付選択フィルター | イベントがある日付マスのみクリック可能。選択状態になり、月次リストを当日のイベントに絞り込み |
| 月次イベントリスト | 当月（または選択日）のイベント一覧を表示、クリックでモーダル表示                             |
| ソート             | 月次イベントリストは startDate 昇順でソート                                                  |

### 2.4 エリアマップ画面（AreaView）

| 機能             | 内容                                                                    |
| ---------------- | ----------------------------------------------------------------------- |
| Leaflet 地図表示 | 群馬県境界（SW: [35.95, 138.25] / NE: [37.05, 139.75]）内に制限         |
| エリアマーカー   | AREA_COORDS で定義した市区町村の座標にマーカー配置                      |
| マーカークリック | クリックすると該当エリアセクションへスムーズスクロール、2秒間ハイライト |
| エリアセクション | EVENTS から一意エリアを抽出し、エリアごとにカード一覧を表示             |
| イベントカード   | EventCard コンポーネントで表示、クリックでモーダル表示                  |

**AREA_COORDS 登録エリア**:  
前橋市、高崎市、桐生市、伊勢崎市、太田市、渋川市、富岡市、安中市、藤岡市、館林市、玉村町、中之条町、みなかみ町、上野村、群馬県（県全体）

### 2.5 情報源画面（SourcesView）

| 機能             | 内容                                                  |
| ---------------- | ----------------------------------------------------- |
| 情報源カード表示 | SOURCES 配列の各情報源をカード形式で表示              |
| 掲載件数表示     | 各情報源（エリア名）の現在の掲載件数を表示            |
| RSS バッジ       | rssUrl が設定されている情報源に「RSS」バッジを表示    |
| 外部リンク       | カレンダー・広報紙・子育て情報の各 URL を別タブで開く |

### 2.7 イベント詳細モーダル（EventModal）

| 機能                  | 内容                                                                                                     |
| --------------------- | -------------------------------------------------------------------------------------------------------- |
| 詳細情報表示          | 会場・日程・種別・対象年齢・費用・説明・タグを表示                                                       |
| 公式サイトリンク      | 別タブで公式サイトを開く（`rel="noopener noreferrer"`）                                                  |

| 閉じる                | オーバーレイクリック、または閉じるボタンで閉じる                                                         |
| provide/inject        | App.vue が `openModal` を provide、各 View が inject して使用                                            |

## 3. イベントデータ管理

### 3.1 ID 体系

| 範囲         | 管理方式                                            |
| ------------ | --------------------------------------------------- |
| 1〜999       | 手動追加（`src/data/events.js` を直接編集）         |
| 10000〜99999 | スクレイパー自動収集（MD5 ハッシュベースの安定 ID） |

### 3.2 カテゴリ

| 値           | 日本語ラベル     |
| ------------ | ---------------- |
| `experience` | 体験・工作       |
| `exhibition` | 展覧会           |
| `nature`     | 自然・アウトドア |
| `culture`    | 文化・学習       |
| `festival`   | 祭り・フェスタ   |

### 3.3 ステータス判定（`getStatus`）

現在日付（深夜 0 時基準）と startDate / endDate を比較して判定する。

| 条件              | ステータス | ラベル   |
| ----------------- | ---------- | -------- |
| 今日 > endDate    | `ended`    | 終了     |
| 今日 >= startDate | `ongoing`  | 開催中   |
| 今日 < startDate  | `upcoming` | 近日開催 |

## 4. スクレイパー概要

### 4.1 実行方式

`node scraper/scrape.js` を手動実行。

### 4.2 対応情報源と取得方式

| 情報源               | 方式                                                    | エリア   |
| -------------------- | ------------------------------------------------------- | -------- |
| 太田市 RSS           | RSS XML                                                 | 太田市   |
| 高崎市カレンダー     | HTML スクレイピング（今月・来月）                       | 高崎市   |
| 前橋市カレンダー     | JSON API（calendar.json）                               | 前橋市   |
| 太田市カレンダー     | HTML スクレイピング                                     | 太田市   |
| 群馬県立自然史博物館 | HTML スクレイピング                                     | 富岡市   |
| 群馬サファリパーク   | HTML スクレイピング                                     | 富岡市   |
| ぐんま天文台         | HTML スクレイピング                                     | 中之条町 |
| ぐんラボ！           | HTML スクレイピング（ページネーション、上限 15 ページ） | 各エリア |

### 4.3 取得期間

実行日から 6 ヶ月先（HORIZON_MONTHS = 6）までのイベントを対象とする。

### 4.4 フィルタリング

子ども関連キーワード（KIDS_KEYWORDS）にタイトル・説明文が一致するイベントのみ取得する。

### 4.5 出力

`src/data/events.js` を上書き更新する。id 1〜999 の手動イベントは保持する。

## 5. データ流れ

```
[scraper/scrape.js]
    │ 各サイトから取得・変換
    ▼
[src/data/events.js] ← 手動イベント（id 1〜999）を保持しつつ id 10000+ を上書き
    │
    ├─ HomeView.vue       読み込み、ステータス別集計・表示
    ├─ EventsView.vue     読み込み、フィルター・ソート・検索
    ├─ CalendarView.vue   読み込み、月・日ごとにフィルター
    ├─ AreaView.vue       読み込み、エリア別に分類・地図マーカー
    └─ SourcesView.vue    読み込み、エリアごとの件数集計
```
